## ADDED Requirements
### Requirement: 项目管理页面
系统 MUST 提供项目管理页面用于展示与管理项目。

#### Scenario: 进入项目管理页面
- **WHEN** 用户访问 `/projects` 路由
- **THEN** 系统应显示项目管理页面
- **AND** 页面应列出所有已存在项目

#### Scenario: 在项目管理页面创建项目
- **WHEN** 用户在项目管理页面选择创建项目
- **THEN** 系统应支持创建本地项目或克隆远程仓库
- **AND** 新项目创建后应出现在项目列表中

#### Scenario: 在项目管理页面更新项目
- **WHEN** 用户在项目管理页面修改项目名称或描述
- **THEN** 系统应保存更新并刷新列表

#### Scenario: 在项目管理页面删除项目
- **WHEN** 用户在项目管理页面删除项目
- **THEN** 系统应将项目从列表移除（不删除本地文件）

### Requirement: 无项目启动锁定
系统 MUST 在应用启动时检查是否存在项目，若无项目则强制跳转并锁定在项目管理页面。

#### Scenario: 启动时无项目
- **WHEN** 应用启动且项目数量为 0
- **THEN** 系统必须导航到 `/projects`
- **AND** 用户在创建项目之前不可访问其他页面

#### Scenario: 创建首个项目后解锁
- **WHEN** 用户在项目管理页面创建至少一个项目
- **THEN** 系统应允许导航到其他页面

#### Scenario: 删除最后一个项目
- **WHEN** 用户删除最后一个项目
- **THEN** 系统应再次锁定并停留在 `/projects`
