# Design: 左边栏布局重构

## Context

当前左边栏设计以任务列表为核心，但随着 VibeWork 功能扩展，需要提供更结构化的功能导航。新设计将左边栏定位为"项目功能导航器"，而非简单的任务列表。

## Goals

- 提供清晰的功能导航结构
- 支持项目级别的上下文切换
- 保持界面简洁，避免信息过载
- 支持折叠/展开两种状态

## Non-Goals

- 不在左边栏直接展示任务详情
- 不实现复杂的项目管理功能（仅切换和创建）

## Design Decisions

### 1. 布局结构

```
┌─────────────────────┐
│  Logo    [折叠按钮] │  <- 顶部固定
├─────────────────────┤
│  ☐ Dashboard        │
│  ☐ 看板             │
│  ☐ Skill            │  <- 功能导航区
│  ☐ MCP              │
├─────────────────────┤
│      (空白区)       │  <- 弹性空间
├─────────────────────┤
│  [当前项目名称]  ▼  │  <- 项目切换器
│  + 新建项目         │
├─────────────────────┤
│  ⚙ 设置             │  <- 设置按钮（最底部）
└─────────────────────┘
```

### 2. 功能导航项

| 导航项 | 图标 | 路由 | 说明 |
|--------|------|------|------|
| Dashboard | LayoutDashboard | /dashboard | 项目概览仪表盘 |
| 看板 | LayoutGrid | /board | 任务看板视图 |
| Skill | Sparkles | /skills | 技能配置 |
| MCP | Server | /mcp | MCP 服务器管理 |

### 3. 项目切换器

- 显示当前选中的项目名称
- 点击展开下拉菜单，显示所有项目
- 提供"新建项目"入口
- 项目切换时更新全局上下文

### 4. 折叠状态

折叠时仅显示图标，悬停显示 tooltip。
